# nidaqmx_on_pi

Small Python package scaffold for NI-DAQmx-related tools on a Raspberry Pi.

Quickstart

1. Create and activate a virtual environment (PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
python -m pip install -r requirements.txt
```

3. Run tests:

```powershell
pytest
```

4. Run the small demo CLI:

```powershell
python -m nidaqmx_on_pi.cli Alice
```
